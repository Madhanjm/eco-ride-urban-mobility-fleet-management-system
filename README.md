# Eco-Ride Urban Mobility System

Eco-Ride is a Python-based project that simulates a simple urban_toggle mobility system.  
This project was built to practice **object-oriented programming, file handling, and unit testing using pytest**.

---

## Project Overview

The system manages different types of vehicles (like electric cars and scooters), stores fleet data, and includes test cases to validate functionality.  
It focuses on clean code structure, modular design, and proper testing.
---

## Features

- Vehicle management using OOP concepts
- Support for multiple vehicle types
- Fleet data stored in CSV and JSON formats
- Controller layer to manage operations
- Unit testing using pytest
- Clean and modular project structure

---
## Technologies Used

- Python
- Pytest
- CSV & JSON file handling
- Object-Oriented Programming
